Ejecución de la Solución

Para levantar el proyectos e debe ejecutar el comando

npm install

Para ejecutar las pruebas del flujo de este primer caso se debe agregar en la carpeta descargas un archivo llamado

Ejemplo.png
Guadado en la ruta:
D:\\Descargas\\Ejemplo.png


Para la ejecución de las pruebas se debe ejecutar:

npx mocha TestCases/Case1.js
